### Writing Principles
- **User-first language**: Simple, clear, friendly
- **Benefit-focused**: Emphasize what users gain
- **Action-oriented**: Clear next steps
- **Scannable format**: Headings, bullets, highlights

- **Color coding**: Consistent use throughout
- **Typography hierarchy**: Clear structure
